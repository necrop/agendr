The GEL2 prototype uses the following technologies:

  • Database: MySQL
  • Web framework: Django 1.5
  • Asynchronous task management (import jobs, etc.): Celery
  • HTML/CSS: Twitter Bootstrap
  • Frequency charts: matplotlib
  • Frequency data is derived from the Google Ngrams corpus.
